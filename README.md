> word文档转换工具

## 安装

`npm install ica-doc -g`

## 使用

```
doc start -------- 自动读取当前文件夹下的doc文件并转换成对应的html

doc start [sorceDir] [distDir] --------- 读取目标文件夹下的doc文件并转换成对应的html
```
